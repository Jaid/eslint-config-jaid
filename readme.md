**Personal ESLint preset. Intentionally ugly to write code that looks like I feel.**


























## Development



Setting up:
```bash
git clone git@github.com:jaid/eslint-config-jaid.git
cd eslint-config-jaid
npm install
```
Testing in production environment:
```bash
npm run test
```
